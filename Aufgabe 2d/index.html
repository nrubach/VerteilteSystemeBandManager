Mit Hilfe dieser Seite können einige Demofunktionen des Bandmanagers abegrufen werden. Die Angezeigten JSON Objekte werden zusätzlich in der Entwicklerkonsole (F12) des Browsers ausgegeben.
Damit diese Demo funktioniert muss die Spring Boot Anwendung aus 2c) auf dem localhost unter Port 8080 aktiv sein. (Wird keine Response angezeigt ist die Spring Anwendung vermutlich nicht aktiv oder falsch eingerichtet.)
